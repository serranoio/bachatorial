# The Strategy

What if you told a baby to do multiplication when they don't even know how to do addition?

This is how everyone, EVERYONE, is with dancing. We see people do multiplication on the dance floor while we don't know how to do basic addition, causing us extreme anxiety.

This problem ends today.

Through my learning approach, ANYONE can become a dancer.

1. Break limiting beliefs. You can dance.
2. Learn the basics of dance.
3. Apply the basics to a dance style. In this case, Bachata
